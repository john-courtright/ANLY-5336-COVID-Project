# ANLY-5336-COVID-Project

Contributors: Authors: John Courtright, Shivani Narang, Dinesh Prabhath

This repository contains files related to TXST's Intro to Analytics (Graduate Level; ANLY 5336) group project. The topic of this project is COVID-19 in Travis County, TX. 

All COVID-19 data comes from The New York Times' COVID data repository. This project uses figures on weekly cases, deaths, and mask use.

The project is split into 3 steps:
1) Exploratory: Parse the data sets to find a relevant county with complete (3-year) data and make any preliminary visualizations
2) Predictive: Construct a time-series model to visualize past trends in cases, hospitalization, and deaths and attempt to forecast future values
